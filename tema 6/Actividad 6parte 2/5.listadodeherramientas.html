• Adobe XD

    Adobe XD es una aplicación de diseño y prototipado que permite crear experiencias

    interactivas y prototipos de alta fidelidad. Es ampliamente utilizado por diseñadores

    de interfaz de usuario y experiencia de usuario.

  • Figma

    Figma es una plataforma de diseño en la nube que facilita la colaboración en tiempo real

    en proyectos de diseño interactivo. Permite crear, prototipar y colaborar en diseño de

    interfaces de usuario.

  • Adobe Creative Cloud

    Adobe Creative Cloud incluye una amplia gama de herramientas, como Photoshop, Illustrator

    y After Effects, que son valiosas para diseñadores interactivos. Estas aplicaciones permiten

    la creación y manipulación de gráficos, animaciones y contenido multimedia.

  • Sketch

    Sketch es una aplicación de diseño vectorial ampliamente utilizada por diseñadores de aplicaciones

    y sitios web. Es conocida por su simplicidad y capacidades de diseño de alta calidad.

  • React

    React es una biblioteca de JavaScript ampliamente utilizada para crear interfaces de usuario

    interactivas y componentes reutilizables. Es especialmente valioso en el desarrollo de aplicaciones

    web de una sola página (SPA).
